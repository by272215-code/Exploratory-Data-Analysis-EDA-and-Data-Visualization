## Introduction

This project focuses on **Exploratory Data Analysis (EDA)** and data visualization using Python. The objective is to understand the dataset, identify patterns, analyze relationships between variables, and present insights through graphical representations.

The project uses a sample student dataset containing information such as age, study hours, and marks. Various statistical and visualization techniques are applied to better understand the data and uncover meaningful trends.

This project helped me gain practical experience in data analysis and visualization, which are essential skills in the field of Data Science.



## Project Objectives

The main objectives of this project are:

* Explore and understand the dataset
* Perform statistical analysis
* Identify relationships between variables
* Visualize data using different charts
* Analyze correlations between features
* Gain insights from the dataset



## Features

✅ Data loading and exploration using Pandas

✅ Dataset information and structure analysis

✅ Statistical summary generation

✅ Correlation analysis

✅ Histogram visualization

✅ Scatter plot visualization

✅ Correlation heatmap visualization

✅ Beginner-friendly implementation



## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn



## Analysis Performed

### 1. Data Exploration

The project begins by examining:

* First few records of the dataset
* Dataset structure and information
* Data types of each column

### 2. Statistical Analysis

Basic statistics are generated, including:

* Mean
* Minimum value
* Maximum value
* Standard deviation
* Percentiles

### 3. Correlation Analysis

A correlation matrix is generated to understand relationships between different variables.

### 4. Data Visualization

Multiple visualizations are created to better understand the dataset:

#### Histogram

Displays the distribution of student marks.

#### Scatter Plot

Shows the relationship between study hours and marks.

#### Correlation Heatmap

Visualizes the strength of relationships among all variables.
